Article

Real-Time Capable System for Hand Gesture Recognition Using Hidden Markov Models in Stereo Color Image Sequences

We developed a real-time recognition system for alphabets (A-Z) and numbers (0-9) using a single hand’s movement trajectory captured in color images. The system employs Hidden Markov Models (HMM) for recognition, with Gaussian Mixture Models (GMM) for skin color detection and the Baum-Welch (BW) algorithm for training. Testing is conducted using advanced algorithms and the Left-to-Right Banded (LRB) Model. Input data comes from videos captured with a Bumblebee stereo camera, resulting in recognition rates of 99.16% during training and 94.72% during testing.
